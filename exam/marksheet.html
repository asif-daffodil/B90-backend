Rubric Reference

Each section carries 4 marks.

1. Project Setup

Node project initialized, required dependencies installed, and nodemon configured.

2. Server & Middleware

Express server, cors, express.json, and dotenv environment usage for values such as PORT or SALT_ROUNDS.

3. Router

Separate router file, POST register and login routes, and router mounted in the main server.

4. Bcrypt

Password hashing during registration and bcrypt.compare during login.

5. Basic Functionality

Temporary user storage in memory, success response on register, and correct success or error handling on login.

Scoring was based on the code as submitted in each folder. Where a project contained the right logic in isolated files but failed to wire files together correctly, marks were deducted in the server or router sections because the expected working output would not be achieved without corrections.
